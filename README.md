# slides

Public publishing target for Founders and Coders teaching decks, built with [Slidev](https://sli.dev).

Source decks are authored in the private `brain` repo; only decks intended for public viewing are mirrored here and deployed to GitHub Pages.

## Live

- Perceptrons: https://sofer.github.io/slides/

## Local

```sh
pnpm install
pnpm dev      # serve the deck at http://localhost:3030
pnpm build    # static build into dist/
```
